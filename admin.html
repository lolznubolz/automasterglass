Управление сайтом AutoMasterGlass

Первый вход — нужен ключ доступа (один раз)

Ключ создаётся бесплатно на GitHub и сохраняется только в этом браузере.

  1. Откройте страницу создания ключа (войдите в свой GitHub).
  2. Token name — любое имя, например admin-site.
  3. Resource owner — ваш аккаунт.
  4. Repository access → Only select repositories → выберите automasterglass.
  5. Permissions → Repository permissions → Contents → Read and write.
  6. Generate token, скопируйте ключ (github_pat_…) и вставьте ниже.

Редактирование сайта

Меняйте, что нужно, и нажмите «Сохранить». Сайт обновится за ~1 минуту.

Контакты

Телефон подставится в кнопку звонка, WhatsApp и по всему сайту.

Цены (в сумах, только числа)

Эти числа автоматически попадают в карточки услуг и в калькулятор — на всех языках.

Галерея работ «До / После»

Добавьте пары фото одного стекла: «До» (со сколом/трещиной) и «После» (после ремонта). Можно несколько пар — посетители будут их листать. Фото сжимаются автоматически. После изменений нажмите «Сохранить» внизу.
