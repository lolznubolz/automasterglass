Управление сайтом AutoMasterGlass

Первый вход — нужен ключ доступа (один раз)

Ключ создаётся бесплатно на GitHub и сохраняется только в этом браузере. Делается один раз.

  1. Откройте страницу создания ключа (нужно войти в свой GitHub).
  2. В поле Token name напишите любое имя, например admin-site.
  3. Resource owner — выберите свой аккаунт.
  4. Repository access → Only select repositories → выберите automasterglass.
  5. Permissions → Repository permissions → найдите Contents → поставьте Read and write.
  6. Нажмите Generate token, скопируйте ключ (начинается на github_pat_…) и вставьте ниже.

Редактирование сайта

Меняйте, что нужно, и нажмите «Сохранить». Сайт обновится за ~1 минуту.
